# ScatterV
ScatterV is a RISC-V processor implemented in Verilog and synthesized on FPGA. It features standard functionality of a RISC-V processor with a custom instruction that loads a pseudorandom number to a register, utilizing primitive polynomials that define a linear feedback shift register (LFSR) to produce the longest sequence possible. The implementation ding hardware level random number generation is foundation for cryptography, simulations, and randomized algorithms.

This repository builds upon my previous project, [https://github.com/LeeT27/learningVerilog](learningVerilog), where I created a simple processor to execute basic ALU, load, store, and jump functions. ScatterV expands beyond this through the integration of RISC-V compatibility, addition of complex instructions, the use of industry standard verification tools, and more.
