# ScatterV
ScatterV is a RISC-V processor implemented in Verilog and synthesized on FPGA. It features standard functionality of a RISC-V processor with a custom instruction that loads a pseudorandom number to a register, utilizing primitive polynomials that define a linear feedback shift register that generate a max length sequence.
